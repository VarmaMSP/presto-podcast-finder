# Podcast Finder
An app to demo the use of [Juspay's Presto framework](https://github.com/juspay/purescript-presto) with [ReactJS](https://github.com/facebook/react)

### Build Instructions 
```
$ bower install 
$ npm install 
$ pulp build
$ npm run build
$ npm run start 
```

#### About Presto 
Presto is a purescript library that allow you to handle things like screen transitions, api calls, state management etc in frontend. At its core is a free monad termed as 'Flow' that enables you to handle frontend logic idiomatically. Docs can be found at [Pursuit](https://pursuit.purescript.org/packages/purescript-presto/0.2.2).
